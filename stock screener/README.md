# Stock Screener Data Pipeline

This project is an automated web scraping pipeline developed as part of my Data Engineering internship.

It extracts financial data from a stock screener website daily, processes it, and stores it locally. The system is containerized with Docker for easy deployment.

## 🛠 Tech Stack
* **Language:** Python 3.11
* **Orchestration:** Prefect
* **Tools:** Playwright, Requests, Pandas
* **Deployment:** Docker & Docker Compose
